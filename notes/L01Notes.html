Git it Together

 

Command line git commands:

 

git status – shows active git actions

git pull – refreshes everything so your code is the same as the current repository

git branch … – Creates a new branch with the given name

git checkout … – Takes you to the specified branch

git branch -a – shows all the active branches

git branch -d … – deletes the specified branch

git push origin –delete … – Deletes a remote branch

git add . – Stages all the code for a commit. The period can be replaced with a desired file

git reset head … – Undos the staging of the specified file.

git commit -m “…” – Creates a new commit that can be merged onto the main branch. “-m” means message.

git reset –soft HEAD~1 – Undos the last commit. You still need to unstage it too.

git push origin master – Pushes the active commit to the main repository.

git revert HEAD – Undos the last push by setting it as a new commit, still needs to be pushed again

git diff – shows all differences between the repository and your current code

git rebase is for personal work and not anything that is shared with others

git clone … – Clones a repository using the link put at the end

git fork … – Clones a repository and puts it on your own github account
