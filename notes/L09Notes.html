HTML5 APIs

 

·      HTML5 brought a lot of new features to html and to web development in general.

·      data- attributes can store html data in elements. Their only use is to be referenced by JavaScript files.

·      They can be left alone or set to a string, and they let us sort through elements easier when using JavaScript.

·      You can store information by setting data-myname = ‘ryan’.

·      You access the information by using element.dataset.myname.

·      Hyphens in the name are replaced with camel case (my-name is myName).

·      HTML5 APIs give you access to information about user hardware.

·      The Web Storage API (window.localStorage) lets you store more info with no expiration. It is a lot easier to use than cookies.

·      localStorage.setItem(‘param’, ‘value’);

·      localStorage.getItem(‘param’); – returns value

·      We can also use it like object properties: –  localStorage.param = ‘value’;

·      localStorage.removeItem(‘param’); – removes an item from storage

·      Adding to storage creates a storage event. Has the parameters key, newValue, oldValue, storageArea

 

‘storage’ event example:

addEventListener('storage', (event) => {

console.log(`The ${event.key} was updated from ${event.oldValue} to ${event.newValue} and saved in ${event.storageArea}`)

}, false);

 

·      The geolocation API gives you access to user location and user movement speed.

·      It needs permissions to function.

·      navigator.geolocation.getCurrentPosition(callbackFunction(position));

·      The callback function can work with the object position, which has the values .speed, .altitude, .heading, .timestamp, .accuracy, .altitudeAccuracy

·      Navigator.geolocation.watchPosition(callback function); – starts a watch position function, which will repeat every time the user changes position.

·      Web Workers are asynchronous scripts that work on things in the background.

·      const worker = new Worker(‘script.js’); – creates a new worker

·      worker.postMessage(‘message’) – Sends information to the worker

 

 

 

‘worker’ event listener:

worker.addEventListener('message', (event) => {

    console.log(event.data);

}, false);

 

·      worker.terminate() – closes the worker from the main script.

·      self.close() – closes the worker from within.

·      The notifications API lets you send notifications on the page.

·      Notification.requestPermission() – requests permission for notifications

·      returns permission, which is either ‘granted’ or ‘denied’

·      notifications can have a body and icon.

 

Example notification:

const notification = new Notification('Notification Name',{

    body: 'Body here',

    icon: 'icon.png'

});

 

·      notification.close() – ends a notification

·      HTML5 now offers use of the <audio> and <video> elements, which uses the src attribute to easily display media elements.

·      Video has the properties play(), pause(), volume, muted, currentTime, playbackRate, loop, duration

·      Events include play, pause, volumechange, loadedmetadata

·      <canvas> lets you draw on the html page

 

Canvas, SVG, and Drag and Drop

 

·      Canvas lets us draw with JavaScript.

·      It needs an id, width, and height

 

var canvas = document.getElementById("myCanvas");

var context = canvas.getContext("2d");

context.strokeStyle = "red";

context.fillStyle = "blue";

context.fillRect(10, 10, 100, 100);  

context.strokeRect(10, 10, 100, 100);

 

·      .fillstyle – selects style of fill

·      context.beginPath() – starts a path

·      context.arc(x, y, radius, startAngle, endAngle, anticlockwise) – makes an arc

·      Canvas can also be used to manipulate images.

·      getImageData() – gets an image that can be messed with.

·      You can also manipulate videos with canvas.

·      SVG is another form of image storage.

·      XML stands for extendible markup language, and we can use it to describe data, like the content of a file

·      You can draw in SVG with HTML5.

·      SVGs have filters that can change their look.

·      Raphael is a JavaScript SVG file creator that makes things a lot easier.

·      Canvas is like Microsoft paint. Once something is drawn, it is a part of the art now. You can’t select or modify anything finalized on a canvas. SVG elements are obtainable through the DOM.

·      <img draggable=‘true’> – makes an image draggable!

·      The element.dataTransfer property lets you select and modify objects as they are dragged.

·      To be able to drop dragged items, we need to set dragOver events (other elements make room) and drop events.
